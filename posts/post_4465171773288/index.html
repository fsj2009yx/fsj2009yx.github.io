参数估计

参数估计问题的定义及其理解

矩估计法

矩估计法（Method of Moments, MoM），也称为数字特征法，是一种常用的参数估计方法。其基本思想是利用样本矩来估计总体矩，即用样本的统计量代替总体的相应统计量进行估计。

样本矩和总体矩

样本矩：

对于一组样本 $X_1, X_2, …, X_n$，样本的第 $k$ 阶矩定义为：

$m_k = \frac{1}{n} \sum_{i=1}^n X_i^k$

总体矩：

对于一个随机变量 X，其理论上的 k 阶矩定义为：

$$μ_k = E(X^k)$$

 

下面我们举例说明：

假设我们有一个正态分布随机变量 $X \sim N(\mu, \sigma^2)$，我们想要估计正态分布的参数 $μ$（均值）和 $\sigma^2$（方差）。

**目标：**我们通过样本来估计以下参数：

  1. 均值 $μ$
  2. 方差 $\sigma^2$

一、样本矩计算

假设我们有一个样本：$X = { X_1, X_2, …, X_n }$

  • 样本均值 $\hat{\mu}$：
$$\hat{\mu} = \frac{1}{n} \sum_{i=1}^n X_i$$
  • 样本方差 $S^2$：
$$S^2 = \frac{1}{n} \sum_{i=1}^n (X_i - \hat{\mu})^2$$

二、总体矩计算

这里我们设总体矩为正态分布，则理论均值为$μ$，理论方差为$ \sigma^2$;

将样本矩和总体矩等式：有$\hat{\mu}=μ$，$S^2≈σ^2$

因此，矩估计法中：

  • 使用样本均值 $\hat{\mu}$ 作为总体均值 $μ$ 的估计。
  • 使用样本方差 $S^2$ 作为总体方差 $\sigma^2$ 的估计。

极大似然估计法

极大似然法是在总体类型已知条件下使用的一种参数估计方法

基本思路：根据样本的具体情况来选择未知参数0的估计量，使样本发生的可能性最大。

似然函数

分为离散型和连续性

离散型总体

设总体X的分布为$P{X=x}=p(x,\theta)$，样本$X_1,X_2…X_n$是来自X的样本，样本观察值为x1,x2…xn

则($X_1,X_2…X_n$)的联合分布律为：

$$P\{X_1=x_1,X_2=X_2,...,X_n=x_n\}=\prod^n_{i=1}p\{x_i;\theta\}$$

1

它是未知参数的函数，记作

$$L(\theta)=L(x_1,x_2…,x_n;\theta)=\prod^n_{i=1}p(x_i;\theta)$$

上面的函数称为样本的似然函数

连续性总体

设总体X的概率密度$f(x;\theta)$，此时样本的似然函数为

$$L(\theta)=L(x_1,x_2…,x_n;\theta)=\prod^n_{i=1}f(x_i;\theta)$$

最大似然估计值和最大似然估计量

区别 最大似然估计量 (MLE) 最大似然值 (Likelihood Value)
定义 参数 $θ$ 使得似然函数最大化 使用最大似然估计量代入似然函数
含义 是一个参数的估计值（如 $\hat{\theta}$ ） 是似然函数的具体数值
目标 求解参数估计值 在给定参数下计算似然函数的值

L(θ)的最大值求法

由微积分学知识，若似然函数L(θ)关于θ有连续偏导数，则最大似然估计0 一般可从方程(组)：

$$\frac{\partial L(θ)}{\partial θ_i}=0，i=1,2,...k$$

又因为lnL(θ)和L(θ)同时取得最大值，等式可转换为：

$$\frac{\partial lnL(θ)}{\partial θ_i}=0，i=1,2,...k$$

接下来是矩估计值的计算：

步骤 1: 样本矩

样本数据为 $x_1 = 1, x_2 = 2, x_3 = 1, x_4 = 3, x_5 = 1$，其样本均值为：$\bar{x} = \frac{1+2+1+3+1}{5} = \frac{8}{5} = 1.6$

步骤 2: 理论矩

理论均值（总体期望）为：

$$E[X]=1⋅P(X=1)+2⋅P(X=2)+3⋅P(X=3)$$$$=>E[X] = 1 \cdot \theta^2 + 2 \cdot 2\theta(1-\theta) + 3 \cdot (1-\theta)^2$$$$=>E[X] = \theta^2 + 4\theta(1-\theta) + 3(1-\theta)^2$$

展开整理：

$$E[X] = \theta^2 + 4\theta - 4\theta^2 + 3 - 6\theta + 3\theta^2$$$$=>E[X] = 3 - 2\theta$$

步骤 3: 矩估计方程

令理论均值等于样本均值：

$$3 - 2\theta = 1.6$$

解方程：

$$2\theta = 3 - 1.6$$$$=>\hat{\theta} =0.7$$

因此，矩估计值为：

$$\hat\theta_\text{MME} = 0.7$$

估计量的评估标准

对同一参数用不同的方法进行估计可能会得到不同的估计量，那么采用哪种估计量为好，这就涉及到用什么样的标准来评价估计量的问题

无偏性

定义

设$\hat\theta=\hat\theta(X_1,X_2,…,X_n)$是未知参数$\theta$的估计量，如果

$$E( \hatθ )=θ$$

则称 $\hat\theta$ 是 $\theta$ 的无偏估计量

  • 记$b_n=E(\hat\theta)-\theta$,称 $b_n$ 为估计量 $\hat \theta$ 的偏差

  • 当$b_n≠0$，称 $\hat\theta$ 是 $\theta$ 的有偏估计

  • 若 $\lim_{x \to \infty} b_n=0$,称 $\hat\theta$ 是 $\theta$ 的渐近无偏估计

性质

不论总体X服从什么分布，当总体的 k 阶矩存在时，样本的 k 阶原点矩 $A_k$ 是总体 k 阶原点矩的无偏估计量。

特别地，样本均值 $\overline{X}$ 是总体均值E(X)的无偏估计量。

(解释：由于样本均值的期望$E(\overline X)$等于总体均值(第六章性质)，如果样本均值也等于总体均值，则 样本均值是总体均值的无偏估计量)

 

有效性

定义

无偏估计并不是唯一的，多个无偏估计之中就存在优劣问题。

我们希望 $\theta$ 的取值能集中于 $\theta$ 附近,而且密集的程度越高越好,方差是描述随机变量取值的集中程度的

区间估计

点估计值仅仅是未知参数的一个近似值，近似程度如何?误差范围多大?可信程度又如何?这些问题是点估计无法回答的。而区间估计正好弥补了点估计的这个缺陷

区间估计要求根据样本给出未知参数的一个范围，并保证参数的真值以指定的较大概率属于这个范围。即以区间的形式给出未知参数的一个范围，同时还给出此区间包含参数真值的可信程度，这种形式的估计称为区间估计

区间估计的概念

定义 设总体$X\sim F(x, \theta)$，$\theta $是未知参数,$X_1,X_2,…,Xn$是来自总体的样本，对于给定值$(0<\alpha <1)$，如果存在两个统计量$\hat\theta_1=\hat\theta_1(X_1,X_2,，…,X_i)$和$\hat\theta_2=\hat\theta_2(X_1,X_2…X_i)$

如果满足

$$P\{\hat\theta_1< \theta <\hat\theta_2\}≥1-a$$

则称随机区间$(\hat\theta_1,\hat\theta_2)$是 $\theta $ 的置信度为 $1-α$ 的置信区间 1-α为置信水平, $\hat\theta_1$ 和 $\hat\theta_2$ 分别称为置信下限和置信上限

评价置信区间好坏标准:

  • 精度: $\hat\theta_2-\hat \theta_1$ 越小越好

  • **置信度:**越大越好

对置信区间的解释

最常出现的对95%置信区间的错误理解：

  • 在95%置信区间内，有95%的概率包括真实参数 （错误！！）

  • 在95%置信区下构建的模型，意味着模型参数对于试图近似的函数有95%的概率是真实值的估计值（错误！！）

正确解释应为：

  • 95%置信区间，意味着如果你用同样的步骤，去选样本，计算置信区间，那么100次这样的独立过程，有95%的概率你计算出来的区间会包含真实参数值，即大概会有95个置信区间会包含真值。

  • 对于某一次计算得到的某一个置信区间，其包含真值的概率，我们无法讨论。

对于未知参数 $\theta$ 的真值，不能说 $\theta$ 以 $1-\alpha $ 的概率落入该随机区间，只能说区间 $(\hat\theta_1,\hat\theta_2)$ 套住了该参数的真值

因为$\theta$ 是一个固定但未知的值，不能将他看作是一个随机变量

两个正态总体参数的置信区间

如果有两个相互独立的正态总体，我们关心的主要是两个总体的均值差和方差比的估计

设$X\sim N(μ_1,\sigma_1^2)，Y\sim N(μ_2,\sigma_2^2)$，$X_1,X_2,…,X_{n1}$ 和 $Y_1,Y_2,…,Y_{n2}$ 分别是总体 X 和 Y 的样本

我们有$\overline X- \overline Y$是$μ_1-μ_2$的无偏点估计

$\sigma_1^2$，$\sigma_2^2$已知

由上，由期望和方差的性质，我们易得 $\overline X-\overline Y \sim N(μ_1-μ_2,\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2})$

两个总体的方差相等，但是方差未知 (即$\sigma_1^2=\sigma_2^2=\sigma^2$，但是$\sigma^2$未知)
